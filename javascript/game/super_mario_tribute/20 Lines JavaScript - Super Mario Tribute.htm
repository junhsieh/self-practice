20 Lines JavaScript Super Mario Tribute

  1. Jump up in the ?-boxes, collect the coins
  2. Stomp all the (harmless) goombas
  3. Exit out the the other pipe
Note: The goombas are not dangerous. You cannot die. In this game, the only winner is YOU! Visit the Games sections for more JavaScript games
For most fun, use with IE7 (or Firefox 3). Left/right arrow to control Mario, up arrow or "a" to jump, down arrow to enter pipe.
Created with 20 "effective" lines of JavaScript (including dynamically generated graphics) and submitted for the Ozone Asylum 20 Lines JavaScript contest.
In the context of these contests, the definition of a "line" is a bit unusual. Please read the blog post for more info about the rules and stuff.
